Customer.io Demo Stores

Real stores.
Real events. Zero setup.

A collection of live demo stores built to test Customer.io features end-to-end. Each store fires real events, identifies users, and supports in-app messages. Point any store at your own workspace in seconds.

How to use
Step 1
Pick a store below
Choose the industry closest to your customer's use case. Each store has a realistic product catalogue and a full event suite already wired up.
Step 2
Connect your workspace
Click Switch workspace in the store banner. Enter your write key and site ID — events flow into your workspace in real time.
Step 3
Build and test
Browse, sign up, place an order. Then build segments, workflows, or full campaign storylines against live data in your own workspace.
Write key — found when you set up a JavaScript source in Data Pipelines. Labelled "API Key" in the source settings.
Site ID — found under Settings → API Credentials → Track API. This is different from your write key.
